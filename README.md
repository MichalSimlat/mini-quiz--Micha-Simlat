# Mini Quiz – Michał Simlat

## Opis projektu

Mini Quiz to prosta aplikacja na Androida, która pozwala użytkownikowi wziąć udział w krótkim quizie składającym się z 5 pytań.  
Aplikacja losuje pytania spośród dostępnej puli, wyświetla je kolejno z trzema wariantami odpowiedzi, a na końcu pokazuje wynik użytkownika.

Quiz zawiera pytania z różnych dziedzin i jest dobrym przykładem nauki tworzenia interaktywnych aplikacji Android w języku Java.

---

## Funkcjonalności

- Losowanie i wyświetlanie 5 pytań na quiz
- Wyświetlanie trzech możliwych odpowiedzi (A, B, C)
- Liczenie poprawnych odpowiedzi i wyświetlanie wyniku na żywo
- Możliwość resetowania quizu i rozpoczęcia od nowa
- Prosty i czytelny interfejs użytkownika

---

## Instrukcja uruchomienia


### Wymagania

- Android Studio (preferowana wersja najnowsza)
- Środowisko Java (JDK) zgodne z Android Studio
